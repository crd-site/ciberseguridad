📖 Guía Completa de Nmap (Kali Linux)

🔍 ¿Qué es Nmap?

Nmap (Network Mapper) es una herramienta de código abierto para exploración de redes y auditoría de seguridad.

Permite:

  • Descubrir hosts activos.
  • Detectar puertos y servicios.
  • Identificar sistemas operativos.
  • Buscar vulnerabilidades con NSE (Nmap Scripting Engine).

👉 Ya viene preinstalado en Kali Linux.

🗂️ Índice de Tipos de Escaneo

  1. Escaneos básicos de puertos TCP/UDP
  2. Escaneos avanzados de puertos (evasión, firewalls, SCTP)
  3. Escaneos de descubrimiento (Host Discovery)
  4. Detección de versiones, SO y traceroute
  5. Escaneo de vulnerabilidades (NSE)
  6. Opciones de rendimiento y control de salida
  7. Exportación de resultados
  8. Referencias

1️⃣ Escaneos básicos de puertos

Escaneo de los 1000 puertos TCP más comunes.

nmap 192.168.1.1

✅ -sS → SYN Scan (sigiloso)

nmap -sS 192.168.1.1

✅ -sT → TCP Connect (completo, más detectable)

nmap -sT 192.168.1.1

✅ -sU → UDP Scan (lento, servicios como DNS/SNMP)

nmap -sU 192.168.1.1

✅ -p → Escaneo de puertos específicos o rangos

nmap -p 22,80,443 192.168.1.1
nmap -p- 192.168.1.1    # todos los puertos

2️⃣ Escaneos avanzados de puertos

✅ -sF / -sN / -sX → FIN / NULL / Xmas Scan

Escaneos evasivos contra firewalls simples.

nmap -sF 192.168.1.1
nmap -sN 192.168.1.1
nmap -sX 192.168.1.1

✅ -sA → ACK Scan (mapear firewalls)

nmap -sA 192.168.1.1

✅ -sW → Window Scan (usa ventana TCP)

nmap -sW 192.168.1.1

✅ -sM → Maimon Scan (FIN+ACK)

nmap -sM 192.168.1.1

✅ -sI zombie_host → Idle Scan (totalmente sigiloso)

nmap -sI 192.168.1.1

✅ -sO → IP Protocol Scan (detecta protocolos IP)

nmap -sO 192.168.1.1

✅ -b ftp_host → FTP Bounce Scan (obsoleto)

nmap -b ftp.example.com 192.168.1.1

✅ -sY → SCTP INIT Scan

nmap -sY 192.168.1.1

✅ -sZ → SCTP COOKIE-ECHO Scan

nmap -sZ 192.168.1.1

3️⃣ Escaneos de descubrimiento (Host Discovery)

✅ -sn → Ping Scan (solo detectar hosts vivos)

nmap -sn 192.168.1.0/24

✅ -Pn → No Ping (asume que todos los hosts están activos)

nmap -Pn 192.168.1.1

✅ Pings TCP/UDP/ICMP

nmap -PS 192.168.1.1   # TCP SYN ping
nmap -PA 192.168.1.1   # TCP ACK ping
nmap -PU 192.168.1.1   # UDP ping
nmap -PE 192.168.1.1   # ICMP echo
nmap -PR 192.168.1.1   # ARP scan (LAN)

✅ -6 → Escaneo IPv6

nmap -6 2001:db8::1

4️⃣ Detección de versiones, SO y traceroute

✅ -sV → Detección de versiones de servicios

nmap -sV 192.168.1.1

✅ -O → Detección de sistema operativo

nmap -O 192.168.1.1

✅ -A → Escaneo agresivo (SO + versiones + traceroute + scripts)

nmap -A 192.168.1.1

✅ --traceroute → Ejecutar traceroute

nmap --traceroute 192.168.1.1

5️⃣ Escaneo de vulnerabilidades (NSE)

✅ --script=vuln → Scripts de vulnerabilidades conocidas

nmap --script=vuln 192.168.1.1

✅ --script=exploit → Scripts de explotación (controlado)

nmap --script=exploit 192.168.1.1

✅ --script=auth → Fallos en autenticación

nmap --script=auth 192.168.1.1

✅ --script=brute → Fuerza bruta sobre servicios

nmap --script=brute 192.168.1.1

✅ Scripts especializados

nmap --script=http-vuln* 192.168.1.1   # vulnerabilidades web
nmap --script=smb-vuln* 192.168.1.1    # vulnerabilidades SMB
nmap --script=malware 192.168.1.1      # detección de malware
nmap --script=safe 192.168.1.1         # solo scripts seguros

6️⃣ Opciones de rendimiento y control de salida

✅ -T0-5 → Timing templates

nmap -T4 192.168.1.1

✅ --top-ports <n> → Escaneo rápido de los puertos más comunes

nmap --top-ports 100 192.168.1.1

✅ --open → Mostrar solo puertos abiertos

nmap --open 192.168.1.1

✅ --reason → Explicar por qué un puerto está en cierto estado

nmap --reason 192.168.1.1

7️⃣ Exportación de resultados

✅ Formatos de salida

nmap -oN salida.txt 192.168.1.1    # Normal (texto)
nmap -oX salida.xml 192.168.1.1    # XML
nmap -oG salida.gnmap 192.168.1.1  # Grepable
nmap -oA resultado 192.168.1.1     # Todos los formatos
