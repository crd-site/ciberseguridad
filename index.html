📂 Pasos para instalar la VM

  1. Descargar y descomprimir
    • Instalá 7-Zip o usá WinRAR
    • Extraé el archivo: 7-Zip > Extraer en 'Win7_x64_86\'
    • Esperá a que finalice la extracción
  2. Abrir la máquina virtual en VMware
    • Abrí VMware Workstation o VMware Player
    • Elegí Open a Virtual Machine
    • Seleccioná WinSevenUltimate.vmx
    • Cuando aparezca el mensaje:
      This virtual machine might have been moved or copied
      Hacé clic en: "I Copied It"
  3. Iniciar sesión
    • Usuario: Admin
    • Contraseña: dejar en blanco (solo presionar Enter)

💾 Requisitos mínimos

  • CPU con virtualización activada (VT-x / AMD-V)
  • 4 GB de RAM libres (recomendado: 8 GB)
  • 20 GB de espacio libre en disco
  • VMware instalado (descargar Player gratuito)
