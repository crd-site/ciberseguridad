✅ Guía: Descarga e Instalación de Kali Linux para VMware

🧾 ¿Qué es Kali Linux?

Kali Linux es una distribución de Linux basada en Debian, especialmente diseñada para realizar pruebas de penetración (penetration testing), auditorías de seguridad, análisis forense digital y evaluación de vulnerabilidades.

Incluye más de 600 herramientas preinstaladas, como Nmap, Wireshark, Metasploit, John the Ripper, Burp Suite, entre muchas otras, que permiten analizar redes, sistemas y aplicaciones desde la perspectiva de un atacante ético.

Kali es utilizado por profesionales de la ciberseguridad, investigadores, instructores y estudiantes para entrenar habilidades ofensivas y defensivas en entornos controlados. Es una herramienta esencial para desarrollar pensamiento crítico, reconocer debilidades y fortalecer la seguridad de infraestructuras tecnológicas.

📄 Documentación oficial: https://www.kali.org/docs/

🤭 Propósito de esta guía

Esta guía te enseña a descargar, importar y comenzar a utilizar Kali Linux en VMware Workstation, utilizando la imagen preconfigurada para laboratorios de ciberseguridad. También se explican las diferencias entre los formatos .ova, .vmx y .iso

🌐 Sitio de descarga oficial

https://www.kali.org

🧰 Paso 1: Descargar Kali para VMware

  1. Ingresá al sitio web https://www.kali.org
  2. Hacé clic en "Download"
  3. Bajá hasta la sección Virtual Machines
  4. Seleccioná Kali Linux VMware 64-bit
  5. Descargá el archivo .7z (imagen comprimida)
  6. Verificá el hash SHA256 (opcional)

📆 Paso 2: Extraer el archivo .7z

  • Usá 7-Zip o WinRAR
  • Extraé el contenido en una carpeta
  • Verás archivos como:
    • Kali-Linux-2025.vmx
    • .vmdk (disco virtual)

⚙️ Paso 3: Importar en VMware

  1. Abrí VMware Workstation
  2. Menu: File → Open
  3. Seleccioná el archivo .vmx desde la carpeta extraída
  4. Asigná un nombre y hacé clic en Finish

🚀 Paso 4: Primer uso

  1. Iniciá la VM: Power on this virtual machine
  2. Credenciales por defecto:
    • Usuario: kali
    • Contraseña: kali
  3. Explorá el sistema o abrí una terminal

💡 Diferencias entre .vmx, .ova y .iso

Formato Descripción Cuándo usarlo
.vmx Archivo de configuración de VM ✅ Recomendado para VMware
.ova Plantilla exportada para múltiples entornos ✅ Ideal para importar en otros hipervisores
.iso Imagen para instalar desde cero ✅ Para instalaciones personalizadas

🧠 Consejos para estudiantes

  • Podés ajustar RAM y CPU desde Edit virtual machine settings
  • Verificá que la red esté en modo NAT o Bridge
  • Tomá snapshots antes de hacer pruebas avanzadas
